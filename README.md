# Online Cake Shop UI

A responsive **online cake shop website design** built using **HTML and CSS only**.
This project focuses on creating a clean, modern, and user-friendly interface for an online cake store.

The website is fully responsive, making it accessible and visually appealing on **desktop, tablet, and mobile devices**.

## Features

* Responsive design for all screen sizes
* Modern and clean user interface
* Cake product catalog layout
* Product detail section
* Navigation menu
* Shopping cart UI layout
* Featured cake section
* Attractive product cards
* Smooth layout using pure CSS
* Mobile-friendly navigation

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Responsive Design (Media Queries)

## Pages / Sections

* Home
* Featured Cakes
* Cake Catalog
* Product Details
* Shopping Cart
* About
* Contact

## Purpose of This Project

This project was created to practice **front-end web development using only HTML and CSS**, focusing on responsive layouts and modern UI design for an e-commerce style website.

## Preview

The website displays a modern cake shop interface where users can browse cakes, view product information, and explore different sections of the store.
